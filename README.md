# ❤️ Heart Failure Prediction Model

## 📌 Overview
This project uses machine learning to predict the risk of heart failure based on patient health data. The goal is to identify patterns in medical features that can help with early detection.

---

## 🎯 Objectives
- Clean and preprocess the dataset
- Explore relationships between health features
- Train a classification model
- Evaluate model performance

---

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn
- Jupyter Notebook

---

## 🔍 Key Insights
- Age and cholesterol levels showed strong correlation with heart disease risk  
- Higher resting blood pressure increased likelihood of heart failure  
- Certain combinations of features improved prediction accuracy  

---

## ⚙️ Model
- Model used: Logistic Regression  
- Train/Test Split: 80/20  
- Evaluation Metrics: Accuracy, Precision, Recall  

---

## 📊 Results
- Accuracy: 0.82
